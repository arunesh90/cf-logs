## Worker
To build the worker, do `LOG_BASE_URL="<url pointing to your log server>" AUTH_KEY="<your auth key>" npm run build` in `worker/`

Example: `LOG_BASE_URL="http://127.0.0.1" AUTH_KEY="test" npm run build`
